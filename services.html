A Mac Mini M4 ($599), a Mac Studio, a Linux server, or a cloud instance (AWS/GCP). We'll advise on the best option for your scale. Most clients start with a Mac Mini — it runs 16 agents comfortably.
What's the ongoing cost after setup?
Typically $150-$300/month for Claude API costs, depending on usage volume. The infrastructure runs on hardware you own — no recurring platform fees. We use tiered LLM routing to keep costs minimal.
Can agents access my existing tools?
Yes. Through MCP (Model Context Protocol), agents can connect to almost any API — CRMs, databases, spreadsheets, email, Slack, project management tools. We set up the integrations as part of the service.
What if I need more agents later?
You'll own the full system and all configuration files. Adding new agents means writing a new soul file and registering it — we include documentation so you can do this yourself. Or we offer add-on agents at $300/each.
Is my data private?
Everything runs on YOUR hardware. No data goes to any third party except the LLM API calls (Anthropic's Claude). Your files, databases, and business data never leave your machine.
